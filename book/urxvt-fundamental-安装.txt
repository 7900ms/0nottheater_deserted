
urxvt

查看区:
/usr/bin/urxvt
/usr/lib/urxvt/perl
urxvt --help 2>&1 | grep options:  #=>没有unicode

配置区:
1.
安装
sudo dnf install rxvt-unicode
/usr/bin/urxvt

2.
测试

urxvt

复制粘贴：
ctrl+alt+c
ctrl+alt+v

3.
命令行参数
urxvt -fn "xft:Ubuntu Mono:size=14"
urxvt -fn "xft:monospace:size=14:antialias=true"
urxvt -fn "xft:Meslo LG S DZ for Powerline:style=Regular:size=14:antialias=true"


4.
配置测试

vi ~/.Xresources
```
URxvt.font: xft:monospace:size=16:antialias=true
```

生效
xrdb ~/.Xresources

成功
