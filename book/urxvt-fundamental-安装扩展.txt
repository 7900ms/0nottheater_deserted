配置区1

~/.Xresources
```
! -- Extensions -- !
URxvt*perl-ext:
URxvt*perl-ext-common: default,selection-to-clipboard,keyboard-select,matcher

URxvt*clipboard.autocopy: true

URxvt.keysym.M-Escape: perl:keyboard-select:activate
URxvt.keysym.M-s: perl:keyboard-select:search

URxvt.underlineURLs:  True
URxvt.url-launcher:   /usr/bin/firefox
URxvt.keysym.M-u:     perl:url-select:select_next
URxvt.matcher.button: 1



```







配置区1
开启默认扩展



```
! Register
URxvt*perl-ext:
URxvt.perl-ext-common: default,keyboard-select,matcher,   selection-to-clipboard

!keyboard-select
URxvt.keysym.M-Escape: perl:keyboard-select:activate
!URxvt.keysym.M-s: perl:keyboard-select:search

!matcher
URxvt.keysym.M-u:     perl:url-select:select_next
URxvt.url-launcher:   /usr/bin/firefox
URxvt.underlineURLs:  True
URxvt.matcher.button: 1

! clipboard
! update clipboard whenever primary gets updated
URxvt.clipboard.autocopy: true
! url-select
! We can middle-click on any url and this will launch the browser defined by `URxvt.url-select.launcher`.
URxvt.keysym.M-u: perl:url-select:select_next
URxvt.url-launcher:   /usr/bin/firefox
URxvt.underlineURLs:  True

```

vi ~/.Xresources
```
URxvt.perl-ext-common: default,selection-to-clipboard,pasta,matcher,keyboard-select
URxvt.keysym.M-u:     perl:url-select:select_next
URxvt.url-launcher:   /usr/bin/firefox
URxvt.underlineURLs:  True
URxvt.matcher.button: 1 
URxvt.keysym.M-Escape:perl:keyboard-select:activate
URxvt.keysym.Control-Shift-V:     perl:pasta:paste
! Comment this if you don't want copy when text is selected
URxvt.clipboard.autocopy: true
```

配置区2

1.
安装
cd /usr/lib/urxvt/perl
sudo wget https://raw.githubusercontent.com/effigies/urxvt-perl/master/fullscreen

https://github.com/bookercodes/awesome-urxvt

2.
vi ~/.Xresources
```
URxvt.perl-ext-common: ...,fullscreen,...
URxvt.keysym.F11: perl:fullscreen:switch
```
